🔗 Connexion GitHub

Pour publier les modifications directement sur votre site, renseignez vos informations GitHub. Ces données sont stockées uniquement dans votre navigateur.

Votre nom d'utilisateur GitHub
Le nom du dépôt GitHub (sans le /)
Créez un token sur GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
Cochez la permission repo (accès complet aux dépôts). Ce token reste dans votre navigateur uniquement.
